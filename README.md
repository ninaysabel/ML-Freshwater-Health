# ML-Freshwater-Health

This project applies machine learning and deep learning techniques to predict chlorophyll-a concentrations, a key indicator of freshwater ecosystem health, using environmental data collected by the U.S. Environmental Protection Agency. This project compares multiple predictive models, including Random Forest, XGBoost, Support Vector Regression, and Neural Networks, to identify the factors most strongly associated with freshwater productivity and support data-driven environmental decision-making
